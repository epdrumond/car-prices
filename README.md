# car-prices

End-to-end project for the extraction, processing and analysis for car prices in Brazil.

**Local discovery and source config:** copy `config/examples/discovery.example.yaml` to `config/local/` (see [PROJECT_OUTLINE.md](PROJECT_OUTLINE.md) Phase 1) and add source-specific field maps and notes there. The `config/local/` directory is listed in `.gitignore` and is not pushed to the remote.
